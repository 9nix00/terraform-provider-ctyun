## Schema

### Required

- `cgw_id` (String) 云网关id
- `cidr` (String) 子网信息
- `ec_id` (String) 云间高速id
- `ip_version` (String) 子网类型。取值范围:ipv4和ipv6
- `rtb_id` (String) 路由表id

### Optional

- `description` (String) 路由描述信息
- `is_black_hole_route` (Boolean) 是否是黑洞路由, 如果选择true，next_hop_type、next_hop_id字段可不填写
- `next_hop_id` (String) 目的实例ID/跨域连接ID，如不是黑洞路由则必填
- `next_hop_type` (String) 下一跳实例的类型，如不是黑洞路由则必填。取值范围：vpc-虚拟私有云，cda-云专线，vpn-vpn网关，cross-跨域连接

### Read-Only

- `create_time` (String) 创建时间，为UTC格式
- `id` (String) 路由规则id
